# 📊 Equity Sales Performance Dashboard

## 🔍 Overview

This project focuses on analyzing equity business performance using SQL and BI dashboards. It provides insights into client activity, trading behavior, and revenue generation.

---

## 🛠 Tools Used

* SQL (Zoho Analytics)
* BI Dashboard (Power BI / Zoho)
* Excel

---

## 📂 Project Files

* SQL Queries for client and trading analysis
* Dashboard for CXO-level insights

---

## 📊 Key Insights

* High number of inactive clients indicating reactivation opportunity
* Strong contribution from top-performing branches
* Revenue trends analyzed across MTD, QTD, and YTD

---

## 📈 Features

* Client lifecycle tracking
* Active vs Inactive analysis
* Revenue & brokerage analysis
* Branch & franchise performance
